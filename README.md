# 使用

## 安装

```bash
git clone https://github.com/michwh/hexo-theme-conscious.git themes/conscious
```

或者直接下载最新源码文件，重命名为conscious放到博客themes目录下

## 使用

修改博客配置文件 `_config.yml` 主题属性 theme 为 `conscious`.

## 更新

```bash
cd themes/conscious
git pull
```
